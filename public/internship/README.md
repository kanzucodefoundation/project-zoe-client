# Project Zoe Internship

Project Zoe is looking for interns across four tracks. Each track has its own one-pager below with the entry bar, trial task, and how work is assigned. Sprints run 4-6 weeks.

- [Coding](./coding.md) — competitive, trial task required
- [QA Testing](./qa.md) — open, low technical barrier
- [Data Clean-up](./data-cleanup.md) — open, low technical barrier
- [User Training](./user-training.md) — open, low technical barrier, high impact on adoption

**Apply here:** https://tally.so/r/BzlLbN

Every track uses the same model: a short trial task decides who's in (not the application form alone), and every intern gets a pre-built backlog of concrete tasks so there's never a "what do I do now" moment.
